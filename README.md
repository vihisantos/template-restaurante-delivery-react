# Template Restaurante & Delivery — React + Vite

Template profissional para restaurantes, hamburguerias, pizzarias e delivery.

## 🚀 Como rodar localmente

```bash
npm install
npm run dev
```

## 🏗️ Build e Deploy

```bash
npm run build
# Deploy no Vercel conectando o GitHub repo
```

## ✏️ Como personalizar

- **Cardápio**: edite o array `menu` em `App.tsx` — adicione quantas categorias e pratos quiser
- **Número WhatsApp**: substitua `55XXXXXXXXXXX` pelo número real com código do país e DDD
- **Fotos dos pratos**: substitua os emojis por `<img src="..." />` dentro dos `.menu-emoji`
- **Google Maps**: na seção de localização, substitua o div `.map-placeholder` por um iframe real

## 📁 Estrutura

```
src/
 ├── App.tsx   ← Cardápio e conteúdo (edite aqui)
 ├── index.css ← Cores e estilos
 └── main.tsx
```
